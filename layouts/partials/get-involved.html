Get Involved

CCC is a great opportunity to present what you have been working on to an engaged audience.

All talks should be submitted to ApacheCon with a note to specify the CCC track.

A CCC flag is being added to the talk submission form, but if you don't see it yet, add a note in the Audience section to specify CloudStack or CCC so the talk ends up in the CCC track.

Each accepted talk includes one free registration to the conference.

The Call For Participation (CFP) closes on February 11th, so get your topic in today.
